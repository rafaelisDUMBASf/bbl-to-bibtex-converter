# BBL to BibTeX Converter

A Python tool to convert LaTeX bibliography files (`.bbl`) to BibTeX format (`.bib`).

## Description

This tool parses LaTeX bibliography files (`.bbl`) generated by LaTeX and converts them to standard BibTeX format (`.bib`). It extracts citation keys, authors, titles, journals, years, and URLs from `.bbl` files and formats them as BibTeX entries.

## Features

- Extracts citation keys from `\bibitem` entries
- Parses author information with proper formatting
- Extracts titles, journals, years, and URLs
- Automatically determines entry type (article or inproceedings)
- Uses appropriate BibTeX fields (journal for articles, booktitle for conferences)
- Handles multiple bibliography entries in a single file
- Command-line interface with flexible options

## Installation

No installation required! This tool uses only Python standard library modules.

## Requirements

- Python 3.x
- Standard library only (no external dependencies)

## Usage

### Command-Line Interface

The converter provides a flexible command-line interface using `argparse`:

#### Basic Usage

```bash
# Convert with explicit input and output files
python bbl-to-bibtex-converter.py input.bbl output.bib

# Convert with automatic output filename (replaces .bbl with .bib)
python bbl-to-bibtex-converter.py example.bbl

# Using flags
python bbl-to-bibtex-converter.py -i input.bbl -o output.bib
python bbl-to-bibtex-converter.py --input example.bbl --output example_output.bib
```

#### Options

- `input` (positional): Input `.bbl` file path
- `output` (positional, optional): Output `.bib` file path. If not provided, defaults to input filename with `.bib` extension
- `-i, --input`: Input `.bbl` file path (alternative to positional argument)
- `-o, --output`: Output `.bib` file path (alternative to positional argument)
- `-v, --version`: Show version information

#### Examples

```bash
# Simple conversion
python bbl-to-bibtex-converter.py example.bbl

# Specify output file
python bbl-to-bibtex-converter.py example.bbl my_bibliography.bib

# Using flags
python bbl-to-bibtex-converter.py -i papers.bbl -o papers.bib

# Get help
python bbl-to-bibtex-converter.py --help
```

### Programmatic Usage

You can also execute the script programmatically:

```python
import subprocess

subprocess.run(['python', 'bbl-to-bibtex-converter.py', 'input.bbl', 'output.bib'])
```

Or import the functions directly (note: rename the file to use underscores if importing as a module):

```python
# If you rename the file to bbl_to_bibtex_converter.py:
from bbl_to_bibtex_converter import main

main('input.bbl', 'output.bib')
```

## File Structure

- `bbl-to-bibtex-converter.py` - Main Python script with CLI interface
- `example.bbl` - Example input file (LaTeX bibliography format)
- `example_output.bib` - Example output file (BibTeX format)

## How It Works

The converter uses regular expressions to:

1. Split the `.bbl` file into individual `\bibitem` entries
2. Extract the citation key from each entry
3. Parse author names, titles, and other metadata
4. Determine entry type (article vs. inproceedings) based on content
5. Format the extracted information as BibTeX entries with appropriate fields
6. Combine all entries into a single `.bib` file

### Entry Type Detection

- **inproceedings**: Detected when the entry contains "In \emph{...}" pattern
- **article**: Default type for other entries, or when journal information is present

### Field Mapping

- Articles use `journal` field for publication venue
- Conference papers use `booktitle` field for venue
- Both types include `author`, `title`, `year`, and optionally `url` fields
